# PC微信无障碍增强

**自本版起不再支持微信3.x，如需微信3.x支持，请下载[1.8.8]版本，并关闭该插件的自动升级。**

### 微信 4.x 包含如下增强

* 支持全局搜索框的搜索结果朗读。
* 自动通知新消息（语音和提示音）
* 引入了一个虚拟消息队列，可以在焦点不离开输入框的情况下逐条查看消息，由于微信端限制，暂无法获得发送者昵称：
   - `Control+Alt+上光标`： 上一条
   - `Control+Alt+下光标`： 下一条
   - `Control+Alt+行首`： 虚拟消息队列中的首条。
   - `Control+Alt+行尾`： 虚拟消息队列中的末条（不一定是最新一条，可以点击“跳转到最新消息“按钮跳转到最新位置，同时刷新虚拟消息队列）。

[1.8.8]: https://github.mirror.nvdadr.com/cary-rowen/WeChatEnhancement/releases/download/1.8.8/WeChatEnhancement-1.8.8.nvda-addon
