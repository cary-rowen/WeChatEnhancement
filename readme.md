# PC微信无障碍增强

* 作者： 沉浮 & Eureka
* 兼容版本： NVDA2021.2 及以上

**注： 使微信支持 NVDA 对象文本查看的思路来源于“NV宝盒”。**

## 增强特性
1. 修复因PC微信无障碍缺陷导致的多处列表项朗读问题，如：
    - @群成员列表；
    - 收藏列表；
    - 消息转发选择联系人列表；
    - 聊天文件列表。
2. 公众号文章留言的朗读支持（需按 NVDA+大键盘8 “打开焦点自动跳转到可聚焦元素”）；
3. F3键： 小程序网页内后退到上一页；
4. F4键： 退出小程序；
5. F3键： 开启/关闭自动朗读新消息（默认关闭）；
6. 消息列表内浏览非普通文本消息时增加音效提示；
7. 支持使用 NVDA 的对象文本查看浏览微信的控件信息。
